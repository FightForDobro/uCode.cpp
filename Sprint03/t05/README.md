# Square Value

### Description

Create a function template `squareValue` that iterates over a container's contents and
squares its values.

### Synopsis
```c++
template <class Container>
void squareValue(Container& container);
```

### Received knowledge
1. [Forward Iterator](https://en.cppreference.com/w/cpp/iterator/forward_iterator)

### How to Build?
```bash
 cmake . -B build && cmake --build ./build
 ```
